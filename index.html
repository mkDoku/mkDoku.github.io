Welcome to my digital home

Hello friend!
Did you ever want to use a library of your favorite programming language, but the tutorials and documentations were outdated, too few, nonexistent or just some minimally documented API references?

If so, this blog is for you.
With my blog posts I want to provide straightforward tutorials about various interesting and useful libraries. For this, Haskell is my programming language of choice, but blog posts in other languages (e.g. C++, Rust, …) will follow.
These blog post will remove some of the frustrations exploring a new library, especially when proper documentation is missing. However, my blog will not solely feature libraries with poor documentation, but also extend the already existing set of tutorials out there. The more the merrier, right?

Most importantly, I am writing the blog posts as I am making myself familiar with these libraries and thus experiencing the troubles of a novice. I am interested in improving both my programming and documentation skills - so, please help and contact me with any kind of suggestions!
